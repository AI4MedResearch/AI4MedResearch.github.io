## Getting started

This is a simplified version of the [al-folio](https://github.com/alshedivat/al-folio) template, focused exclusively on providing a high-quality technical blogging experience with Distill-style layouts. All academic-specific features (CV, Publications, Projects, etc.) have been disabled to provide a distraction-free writing environment.

For quick setup, see [QUICKSTART.md](QUICKSTART.md).

## Features

### The iconic style of Distill
The theme allows you to create blog posts in the [distill.pub](https://distill.pub/) style. For more details on how to create distill-styled posts using `<d-*>` tags, please refer to [the example](https://AI4MedResearch.github.io/blog/2021/distill/).

### Full support for math & code
Full support for fast math typesetting through [MathJax](https://www.mathjax.org/) and code syntax highlighting using [GitHub style](https://github.com/jwarby/jekyll-pygments-themes). Also supports [chartjs charts](https://www.chartjs.org/), [mermaid diagrams](https://mermaid-js.github.io/mermaid/#/), and [TikZ figures](https://tikzjax.com/).

### Photos, Audio, and Video
Photo formatting is made simple using [Bootstrap's grid system](https://getbootstrap.com/docs/4.4/layout/grid/), with support for [video](https://AI4MedResearch.github.io/blog/2023/videos/) and [audio](https://AI4MedResearch.github.io/blog/2023/audios/) embeds.

### Light/Dark Mode
Built-in light/dark mode that automatically detects the user's preferred color scheme or can be switched manually.
